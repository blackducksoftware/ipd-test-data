Here is a Python program that generates the first 20 prime numbers:

```python
def is_prime(n):
    if n <= 1:
        return False
    if n <= 3:
        return True
    if n % 2 == 0 or n % 3 == 0:
        return False
    i = 5
    while i * i <= n:
        if n % i == 0 or n % (i + 2) == 0:
            return False
        i += 6
    return True

count = 0
num = 2
while count < 20:
    if is_prime(num):
        print(num)
        count += 1
    num += 1
```

This program defines a function `is_prime` to check if a number is prime or not. It then loops through numbers starting from 2 and checks if each number is prime. If a number is prime, it is printed and the count is incremented until we have printed the first 20 prime numbers.
