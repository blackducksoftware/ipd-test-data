Certainly! Here's a Python function that achieves this:

```python
def unique_elements(arr1, arr2):
    unique_set = set(arr1 + arr2)
    unique_arr = list(unique_set)
    return unique_arr

# Example usage
arr1 = [1, 2, 3, 4]
arr2 = [3, 4, 5, 6]
result = unique_elements(arr1, arr2)
print(result)  # Output: [1, 2, 3, 4, 5, 6]
```
