```python
def fibonacci(n):
    fib_list = [0, 1]
    for i in range(2, n):
        fib_list.append(fib_list[i-1] + fib_list[i-2])
    return fib_list

n = 20
fibonacci_list = fibonacci(n)
print(fibonacci_list)
```

This program defines a function `fibonacci` that generates the first `n` Fibonacci numbers and then calls this function to generate the first 20 Fibonacci numbers. Finally, it prints the list of Fibonacci numbers to the console output.
