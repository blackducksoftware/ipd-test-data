Sure! Here is a Python program that generates a list of prime numbers up to a certain integer input:

```python
def is_prime(num):
    if num < 2:
        return False
    for i in range(2, int(num**0.5) + 1):
        if num % i == 0:
            return False
    return True

def generate_prime_list(limit):
    prime_list = []
    for i in range(2, limit + 1):
        if is_prime(i):
            prime_list.append(i)
    return prime_list

limit = int(input("Enter an integer: "))
prime_numbers = generate_prime_list(limit)
print("List of prime numbers up to", limit, "is:", prime_numbers)
```

You can run this program and enter an integer input to get a list of prime numbers up to that integer.
