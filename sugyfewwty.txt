Here is a Python program that implements the prompt:

```python
def is_prime(num):
    if num < 2:
        return False
    for i in range(2, int(num**0.5) + 1):
        if num % i == 0:
            return False
    return True

def filter_prime_numbers(input_list):
    prime_list = []
    for num in input_list:
        if is_prime(num):
            prime_list.append(num)
    return prime_list

# Input list of numbers
numbers = [2, 3, 5, 6, 8, 11, 13, 17, 19, 22, 23, 29]

# Filter prime numbers from input list
prime_numbers = filter_prime_numbers(numbers)

# Print the prime numbers
print("Prime numbers in the input list:", prime_numbers)
```
