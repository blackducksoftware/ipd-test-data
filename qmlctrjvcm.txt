Here is a Python program that generates a list of prime numbers up to a given integer input:

```python
def is_prime(num):
    if num < 2:
        return False
    for i in range(2, int(num ** 0.5) + 1):
        if num % i == 0:
            return False
    return True

def generate_prime_list(n):
    prime_list = []
    for i in range(2, n+1):
        if is_prime(i):
            prime_list.append(i)
    return prime_list

n = int(input("Enter an integer: "))
prime_list = generate_prime_list(n)
print("Prime numbers up to", n, "are:")
print(prime_list)
```

You can run this program and enter an integer input, and it will generate a list of prime numbers up to that input.
