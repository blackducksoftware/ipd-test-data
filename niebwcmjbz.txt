Here is the Python code that calculates the Fibonacci sequence up to a given number:

```python
def fibonacci(n):
    fib_list = [0, 1]
    while fib_list[-1] + fib_list[-2] <= n:
        fib_list.append(fib_list[-1] + fib_list[-2])
    return fib_list

# Get user input for the number up to which to calculate the Fibonacci sequence
num = int(input("Enter a number: "))

# Call the fibonacci function and print the result
result = fibonacci(num)
print("Fibonacci sequence up to", num, ":", result)
```

You can run this code in a Python environment, enter a number when prompted, and it will generate the Fibonacci sequence up to that number.
