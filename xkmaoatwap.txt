```python
def get_even_numbers(input_list):
    """
    This function takes in a list of numbers and returns a list of only the even numbers in the original list.
    
    Parameters:
    input_list (list): A list of numbers
    
    Returns:
    list: A list of only the even numbers in the original list
    """

    even_numbers = []

    for num in input_list:
        if isinstance(num, (int, float)):
            if num % 2 == 0:
                even_numbers.append(num)
        else:
            return "Error: Non-numeric inputs found in the list"

    return even_numbers

# Test the function
input_list = [1, 2, 3, 4, 5, 'a']
result = get_even_numbers(input_list)
print(result)
```

When you run this program with the provided `input_list`, it will check each element in the list and return a new list containing only the even numbers. If any non-numeric inputs are found, it will return an error message. You can customize the `input_list` to test the function with different lists of numbers.
