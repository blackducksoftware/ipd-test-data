```python
def sum_of_even(numbers):
    even_sum = sum([num for num in numbers if num % 2 == 0])
    even_numbers = [num for num in numbers if num % 2 == 0]
    return even_sum, even_numbers

# Taking input from the user
input_numbers = input("Enter a list of integers separated by spaces: ")
numbers = [int(num) for num in input_numbers.split()]

even_sum, even_numbers = sum_of_even(numbers)

print(f"Sum of even numbers in the list: {even_sum}")
print("List of even numbers from the input list:")
print(even_numbers)
```

This code snippet defines a function `sum_of_even` that takes a list of integers as input, calculates the sum of all even numbers in the list, and generates a new list containing only the even numbers using list comprehensions. The input list is taken from the user and processed accordingly to produce the desired output.
