Here is a sample code snippet in Python that implements a basic calculator program with error handling:

```python
def calculator():
    try:
        num1 = float(input("Enter the first number: "))
        num2 = float(input("Enter the second number: "))
        operator = input("Enter the operator (+, -, *, /): ")

        if operator == '+':
            result = num1 + num2
        elif operator == '-':
            result = num1 - num2
        elif operator == '*':
            result = num1 * num2
        elif operator == '/':
            if num2 == 0:
                raise ZeroDivisionError
            result = num1 / num2
        else:
            raise ValueError

        print(f"Result: {num1} {operator} {num2} = {result}")

    except ZeroDivisionError:
        print("Error: Cannot divide by zero")
    except ValueError:
        print("Error: Invalid operator")
    except Exception as e:
        print(f"An error occurred: {e}")

calculator()
```

You can copy and paste this code into a Python interpreter or IDE to run the program. It will prompt the user to enter two numbers and an operator, then display the result of the calculation. It includes error handling for division by zero and invalid inputs. Feel free to customize or enhance the code as needed!
