Prompt: Implement a function that takes in a list of integers and returns the sum of all the even numbers in the list.

LLM-generated code in Python:

```python
def sum_even_numbers(nums):
    total = 0
    for num in nums:
        if num % 2 == 0:
            total += num
    return total
```

```java
public static int sumEvenNumbers(int[] nums) {
    int total = 0;
    for (int num : nums) {
        if (num % 2 == 0) {
            total += num;
        }
    }
    return total;
}
```
