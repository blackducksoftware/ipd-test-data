Prompt: Write a program that calculates the sum of all even numbers within a given range using <programming language of choice> and outputs the result.

LLM Response in Python:

```
def sum_even_numbers(start, end):
    total = 0
    for num in range(start, end+1):
        if num % 2 == 0:
            total += num
    return total

start_range = int(input("Enter the starting number of the range: "))
end_range = int(input("Enter the ending number of the range: "))

result = sum_even_numbers(start_range, end_range)
print(f"The sum of even numbers between {start_range} and {end_range} is: {result}")
```
