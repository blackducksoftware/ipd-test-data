```python
def is_prime(num):
    if num < 2:
        return False
    for i in range(2, int(num**0.5)+1):
        if num % i == 0:
            return False
    return True

prime_numbers = []
for i in range(1, 101):
    if is_prime(i):
        prime_numbers.append(i)

print("Prime numbers between 1 and 100:")
print(prime_numbers)
```

This program defines a function `is_prime` that checks if a given number is prime. It then generates a list of prime numbers between 1 and 100 by iterating through the numbers and checking if each number is prime. Finally, it prints out the list of prime numbers.
