Sure, here is an example program in Python that takes an input number from the user and outputs all the prime numbers up to that number:

```python
def is_prime(num):
    if num < 2:
        return False
    for i in range(2, int(num**0.5) + 1):
        if num % i == 0:
            return False
    return True

def get_primes_up_to_num(num):
    prime_numbers = []
    for i in range(2, num + 1):
        if is_prime(i):
            prime_numbers.append(i)
    return prime_numbers

input_num = int(input("Enter a number: "))
prime_numbers = get_primes_up_to_num(input_num)

print("Prime numbers up to", input_num, "are:")
print(prime_numbers)
```

You can copy and paste this code into a Python interpreter, and it will prompt you to enter a number. It will then output all the prime numbers up to that number.
