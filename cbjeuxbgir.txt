Here is a sample program in Python that checks if a given number is a prime number or not:

```python
def is_prime(num):
    if num <= 1:
        return False
    for i in range(2, int(num**0.5) + 1):
        if num % i == 0:
            return False
    return True

num = int(input("Enter a number: "))
if is_prime(num):
    print(num, "is a prime number")
else:
    print(num, "is not a prime number")
```

You can run this program in any Python compiler or interpreter. Simply enter a number and the program will output whether it is a prime number or not.
