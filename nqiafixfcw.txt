Here is the Python code that implements the Fibonacci program based on the provided prompt:

```python
def fibonacci(n):
    if n <= 0:
        return []
    elif n == 1:
        return [0]
    elif n == 2:
        return [0, 1]
    else:
        fib_list = fibonacci(n-1)
        fib_list.append(fib_list[-1] + fib_list[-2])
        return fib_list

n = int(input("Enter a number n: "))
fibonacci_numbers = fibonacci(n)
print(f"The first {n} Fibonacci numbers are: {fibonacci_numbers}")
```
