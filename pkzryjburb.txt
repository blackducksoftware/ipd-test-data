Sure, here is a Python program that generates the first 20 Fibonacci numbers and prints them out in a comma-separated list:

```python
def fibonacci(n):
    fib_list = [0, 1]
    for i in range(2, n):
        fib_list.append(fib_list[i-1] + fib_list[i-2])
    return fib_list

n = 20
fib_numbers = fibonacci(n)
print(','.join(map(str, fib_numbers)))
```

This program defines a function `fibonacci` that generates the first `n` Fibonacci numbers and then prints them out in a comma-separated list using the `join` method.
