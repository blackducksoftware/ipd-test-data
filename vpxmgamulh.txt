Here is a Python program that calculates the nth Fibonacci number using dynamic programming and includes error handling for non-negative integer input:

```python
def fibonacci(n):
    if n <= 0:
        return "Invalid input. Please enter a non-negative integer."
    
    fib = [0, 1]
    for i in range(2, n+1):
        fib.append(fib[i-1] + fib[i-2])
    
    return fib[n]

try:
    n = int(input("Enter a non-negative integer value for n: "))
    result = fibonacci(n)
    print(f"The {n}th Fibonacci number is: {result}")
except ValueError:
    print("Invalid input. Please enter a non-negative integer value.")
```

You can copy and paste this code into a Python environment (like an IDE or online interpreter) to run the program. It will prompt the user to enter a non-negative integer value for n and output the nth Fibonacci number. If the user enters a negative number or a non-integer value, it will display an error message.
