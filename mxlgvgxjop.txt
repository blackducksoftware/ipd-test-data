```python
def is_prime(n):
    if n < 2:
        return False
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:
            return False
    return True

for i in range(1, 101):
    if is_prime(i):
        print(i)
```

This Python program defines a function `is_prime` that checks if a number is prime. It then iterates over the numbers from 1 to 100 and prints out the prime numbers using the `is_prime` function.
