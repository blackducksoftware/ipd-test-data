Here's an example code that implements the bubble sort algorithm in Python to sort a list of integers in descending order:

```python
def bubble_sort_descending(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] < arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr

# Test the function with a sample list
arr = [64, 34, 25, 12, 22, 11, 90]
sorted_arr = bubble_sort_descending(arr)
print("Sorted array in descending order:", sorted_arr)
```

When you run this code, it will output:

```
Sorted array in descending order: [90, 64, 34, 25, 22, 12, 11]
```
