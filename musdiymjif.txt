```python
from typing import List

def remove_duplicates(input_list: List[int]) -> List[int]:
    unique_values = []
    seen = set()
    
    for num in input_list:
        if num not in seen:
            unique_values.append(num)
            seen.add(num)
    
    return unique_values

# Test the function
```

This function loops through the input list, checks if the number has been seen before using a set `seen`, and if not, adds it to the `unique_values` list. Finally, it returns the list of unique values. You can test the function with the provided example input list.
